Quickstart

CGO not required — it compiles to one static binary.

shell
# build — one static binary, no dependencies
go build -o otelstore ./cmd/otelstore

# run, persisting to a local file
./otelstore -db-path ./telemetry.db

# now send OTLP to :4317 (gRPC) or :4318 (HTTP), query on :4319

Four ports, one process

Use with Claude Code

Claude Code emits OTLP by default. Point it here and query what it reports.

~/.bashrc · env
export CLAUDE_CODE_ENABLE_TELEMETRY=1
export OTEL_METRICS_EXPORTER=otlp
export OTEL_LOGS_EXPORTER=otlp
export OTEL_EXPORTER_OTLP_PROTOCOL=grpc
export OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:4317
then query it
curl "localhost:4319/v1/metrics?name=claude_code.cost.usage"
